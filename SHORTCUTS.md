# Keyboard Shortcuts

Shortcuts fire when focus is not inside a text input, select, or textarea.
Modifier keys (Ctrl, Cmd, Alt) are ignored — plain key only.

## Navigation

| Key | Destination |
|-----|-------------|
| `H` | Dashboard (Home) |
| `W` | Week view |
| `M` | Month view |
| `Y` | Year view |
| `Q` | Quarter view |
| `E` | Entry (log data) |
| `Esc` | Close dropdowns / menus |
